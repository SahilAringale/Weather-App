# Weather App

This is a simple weather forecast application made using **HTML**, **CSS**, and **JavaScript**.  
It shows the current temperature and weather condition of any city using the **OpenWeatherMap API**.

## Features
- Enter any city name to get weather details  
- Shows temperature in Celsius  
- Displays weather description  
- Simple and clean design  

## How to Run
1. Download or clone this repository  
2. Open the `index.html` file in your browser  
3. Enter a city name and click **Get Weather**

## API Used
- OpenWeatherMap API (https://openweathermap.org/api)
